Competitions

  • Competitions by sport

    • List of open competitions for a given sport.
    • Returns all countries by default.
    GET
    GET /sports/1/competitions
    Request
    headers curl 
    accept: application/json

    curl -X GET -H "accept: application/json" "http://localhost/sports/1/competitions"
    Response 200 OK
    headers body 
    content-type: application/json; charset=utf-8

    {
  "competitions": [
    {
      "id": 1,
      "country": "gb",
      "name": "English premier league"
    },
    {
      "id": 2,
      "country": "fr",
      "name": "Coupe de France"
    }
  ]
}

  • Filter by country

    • Only get competitions for a given country
    • Uses the official ISO country codes
    GET
    GET /sports/1/competitions?country=fr
    Request
    headers curl 
    accept: application/json

    curl -X GET -H "accept: application/json" "http://localhost/sports/1/competitions?country=fr"
    Response 200 OK
    headers body 
    content-type: application/json; charset=utf-8

    {
  "competitions": [
    {
      "id": 2,
      "country": "fr",
      "name": "Coupe de France"
    }
  ]
}
